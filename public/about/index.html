About Us

This project started from 7-1 Sass Architecture and Sass Guidelines while eating oatcakes. And so it was decided to name the project Oatcake. Naturally, it only made sense to design Oatcake to be used with JAM. And here is that attempt. Made with love by Amply LLC

What It Does

Oatcake is a component library similar to Bootstrap. Responsive components customizable through classes.

Our Mission

Oatcake aims to help others develop quicker and better websites and to bring websites to those in need.
